# User Study Documents

This directory contains the documents that were used to conduct the user study.

## Folder Structure


- [`information_sheet.pdf`](./information_sheet.pdf) — Provided to participants before participation; explains the study's purpose, procedures, and data handling
- [`consent_data_processing.pdf`](./consent_data_processing.pdf) — Consent form authorizing the processing of participants’ personal data
- [`consent_research_participation.pdf`](./consent_research_participation.pdf) — Consent form confirming voluntary participation in the research
- `questionnaire_*.pdf` — Pre-study, post-level, and final questionnaires used to collect participants' perception
- [`participant_mapping.pdf`](./participant_mapping.pdf) — Internal mapping sheet used to link participant IDs to anonymized responses
